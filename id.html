ID

<

>

→ Využivajú sa na definovanie prvkov v CSS

→ Využivajú sa na definovanie prvkov v JavaScripte

Id atribút

→ id atribút možno dať každému začiatočnému tagu

//index.html

<a id="my-link" href="index.html" >Navštív HTML kurz</a>

//styles.css

#my-link {
    color: red;
    text-align: right;
}

//class v CSS vyjadrujeme mriežkou (#) pred názvom triedy

❌ Id atribút

→ musí byť jedinečný a nesmú ho mať žiadne dva prvky rovnaký

//index.html

<a id="main" href="index.html" >Navštív HTML kurz</a>
<p id="main">Navštív HTML kurz</p>

Záložka

→ Ak pridáme id prvku, tak sa naň môžeme odvolávať odkazom

→ Dokument musí byť rozsiahlejší, aby sme videli, že nás daný link naozaj odkazuje na prvok s id

//index.html

<h2 id="C4">HTML Tutorial</h2>
