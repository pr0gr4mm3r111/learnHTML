HTML Style atribút

→ Definuje štýly prvku, na ktorom je umiestený

→ style atribút má následovnú syntax (zápis/skladba)

//index.html

<tagname style="vlastnosť:hodnota;">

Background Color

→ CSS background-color definuje farbu pozadia

//index.html

<p style="background-color:red; color:white;">Tento paragraf má červenú farbu pozadia a biely text</p>

//output

Tento paragraf má červenú farbu pozadia a biely text

Text Color

→ CSS color definuje farbu textu

//index.html

<p style="color:white;">Tento paragraf má biely text</p>

//output

Tento paragraf má biely text

Fonts

→ CSS font-size definuje veľkosť textu

→ CSS font-family definuje štýl textu

→ CSS text-align definuje či je text napravo, naľavo alebo v strede

//index.html

<p style="font-size:white;">Tento paragraf má biely text</p>
<p style="font-family:white;">Tento paragraf má biely text</p>
<p style="text-align:(right/left/center);">Tento paragraf má biely text</p>

//output

Tento paragraf je väčší

Tento paragraf je písaný iným štýlom

Tento paragraf je v strede

Zhrnutie

  • style definuje štýly prvku
  • background-color definuje farbu pozadia
  • color definuje farbu textu
  • font-family definuje štýl textu
  • font-size definuje veľkosť textu
  • text-align definuje či je text napravo, naľavo alebo v strede
