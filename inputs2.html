HTML Input → Atribúty

<

>

Readonly atribút

→ Pri použití tohto atribútu nedokážeme do daného inputu zapisovať iba čítať

//index.html

  <input readonly type="text" id="myText" name="text">

Disabled atribút

→ Pri použití tohto atribútu sa daný input nedá používať

//index.html

  <input disabled type="text" id="myText" name="text">

Size atribút

→ Pri použití tohto atribútu určujeme počet znakov, ktorý sa do inputu vôjde

//index.html

  <input size="50" type="text" id="myText" name="text">

Maxlength atribút

→ Pri použití tohto atribútu určujeme najdlhší možný vstup

//index.html

  <input maxlength="4"size="4" type="text" id="myText" name="text">

min a max atribúty

→ Pri použití týchto atribútov určujeme minimum a maximum pre vstup

//index.html

  <input min="18" max="25"size="4" type="number">

Placeholder atribút

→ Pri použití týchto atribútov určujeme prednastavenú hodnotu

//index.html

  <input placeholder="hello" type="text">

Required atribút

→ Pri použití tohoto atribútu určujeme, že daný input musí byť vyplnený, aby sa údaje mohli odoslať

//index.html

<form>
  <label for="username">Username:</label>
  <input type="text" id="username" name="username" required>
</form>

List atribút

//index.html

<form>
  <input list="browsers">
  <datalist id="browsers">
    <option value="Internet Explorer">
    <option value="Firefox">
    <option value="Chrome">
    <option value="Opera">
    <option value="Safari">
  </datalist>
</form>

//output

Autocomplete atribút

→ Pri použití tohoto atribútu prehliadač ponúka hodnoty inputu ak sme v minulosti už nejaké údaje zadávali

//index.html

<input type="text" id="username" name="username" autocomplete="on">

//index.html
