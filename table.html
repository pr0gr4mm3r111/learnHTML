Tabuľky

<

>

→ Tabuľky umožňujú programátorovi vkladať dáta do stĺpcov a riadkov

Tabuľka

<table> definuje tabuľku na HTML stránke

//index.html

<table>
...
</table>

Bunka tabuľky

<td> definuje bunku tabuľky

//index.html

<table>
    <tr>
        <td>Bunka</td>
    </tr>
</table>

Riadok tabuľky

<tr> definuje riadok tabuľky

//index.html

<table>
    <tr>
        <td>Bunka</td>
    </tr>
</table>

Hlavička tabuľky

<tr> definuje hlavičku tabuľky

//index.html

<table>
    <tr>
        <th>Bunka</th>
    </tr>
</table>

Pridanie ohraničenia tabuľky

//styles.css

table, th, td {
   border: 1px solid black;
   border-collapse: collapse;
}

❕ border-collapse je nevyhnutné pridať, inak vzhľad tabuľky bude iný
