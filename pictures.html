Obrázky

<

>

→ obrázky môžu skrášliť danú stránku alebo ilustrovať text

Syntax

//index.html

<img src="url" alt="alternate text" >

Alt atribút

<img> definuje obrázok a atribút alt definuje text, ktorý sa na stránke zobrazí ak nastane problém s načítaním obrázka

Obsah alt atribútu by mal vystihovať to, čo je na obrázku

Src atribút

<img> definuje obrázok a atribút src = " " definuje cestu k danému obrázku

  1. Absolútna cesta k obrázku - src="https://www.w3schools.com/images/img_girl.jpg"
  2. Relatívna cesta k obrázku - src="/images/img_girl.jpg"

Background-image HTML stránky

→ background-image → definuje obrázok ako pozadie daného prvku

→ background-repeat → keď je obrázok malý, tak ho prehliadač bude zobrazovať viac krát

→ background-attachment → keď je stránka väčšia a scrolluje sa môžeme upraviť, aby sa obrázok nescrolloval

→ background-size → nastavuje veľkosť použitého obrázku

//styles.css

    body {
        background-image: url('img_girl.jpg');
        background-repeat: no-repeat;
        background-attachment: fixed;
        background-size: cover;
    }
