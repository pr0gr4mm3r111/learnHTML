Zoznamy

<

>

Typy zoznamov:

  • Očíslovaný zoznam
  • Neočíslovaný zoznam
  • Opisný zoznam

Neočíslovaný zoznam

Neočíslovaný zoznam sa začína <ul> tagom

Každý prvok zoznamu začína <li> tagom

Pri neočíslovanom zozname môžeme pomocou CSS nastaviť:

  • list-style-type:disc
  • list-style-type:circle
  • list-style-type:square
  • list-style-type:none

//index.html

<head>
<style>
  ul {
    list-style-type: square
  }
</style>
</head>
<body>
<ul>
   <li>Káva</li>
   <li>Čaj</li>
   <li>Mlieko</li>
</ul>
</body>

//output

  • Káva
  • Čaj
  • Mlieko

Očíslovaný zoznam

Očíslovaný zoznam sa začína <ol> tagom

Každý prvok zoznamu začína <li> tagom

Pri očíslovanom zozname môžeme v začiatočnom tagu pridať atribút:

  • type="1" → zoznam bude očívlovaný: {1, 2, 3,...}
  • type="A" → zoznam bude očívlovaný: {A, B, C,...}
  • type="a" → zoznam bude očívlovaný: {a, b, c,...}
  • type="I" → zoznam bude očívlovaný: {I, II, III,...}
  • type="i" → zoznam bude očívlovaný: {i, ii, iii,...}

//index.html

<ol type="A">
   <li>Káva</li>
   <li>Čaj</li>
   <li>Mlieko</li>
</ol>

//output

  1. Káva
  2. Čaj
  3. Mlieko

Môžete nastaviť číslo, od ktorého bude zoznam počítať:

//index.html

<ol start="50">
   <li>Káva</li>
   <li>Čaj</li>
   <li>Mlieko</li>
</ol>

//output

  1. Káva
  2. Čaj
  3. Mlieko

Opisný zoznam

Opisný zoznam sa začína <dl> tagom

Každý prvok zoznamu začína <dt> alebo <dd> tagom

<dt> = názov
<dd> = opis

//index.html

<dl>
   <dt>Káva</dt>
   <dd>- čierny horúci nápoj</dd>
   <dt>Mlieko</dd>
   <dd>- biely studený nápoj</dd>
</dl>

//output

Káva
- čierny horúci nápoj
Mlieko
- biely studený nápoj
