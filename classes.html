Triedy

<

>

→ Využivajú sa na definovanie prvkov v CSS

→ Využivajú sa na definovanie prvkov v JavaScripte

Class atribút

→ class atribút možno dať každému začiatočnému tagu

//index.html

<a class="my-link" href="index.html" >Navštív HTML kurz</a>

→ triedu v CSS vyjadrujeme bodkou (.) pred názvom triedy

//styles.css

.my-link {
    color: red;
    text-align: right;
}

Viaceré class atribúty

→ Prvok môže mať viac tried naraz (musia byť oddelené medzerou)!

//index.html

<a class="my-link main" href="index.html" >Navštív HTML kurz</a>

//styles.css

.my-link {
    color: red;
    text-align: right;
}
.main {
    font-family: sans-serif
    font-size: 2rem;
}

Class atribút môžu zdieľať viaceré prvky

//index.html

<a class="main" href="index.html" >Navštív HTML kurz</a>
<p class="main" >Navštív HTML kurz</p>
